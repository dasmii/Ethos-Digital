# Ethos Digital

## Sobre o Projeto
Este projeto foi desenvolvido ao longo das aulas de **Java** como trabalho de fim de período.  
O tema central é **Ethos Digital**, que busca promover **educação cidadã e ética digital**, abordando problemáticas como **cyberbullying** e **discurso de ódio**.

O sistema foi construído utilizando **Spring Boot** e aplica os pilares da **Orientação a Objetos** para criar um software funcional de ponta a ponta.

---

## Objetivos
- Conscientizar sobre comportamentos nocivos no ambiente digital.
- Oferecer cursos e campanhas educativas.
- Simular situações práticas de cyberbullying e discurso de ódio.
- Emitir certificados digitais após conclusão de cursos.
- Gerar relatórios automáticos de participação e impacto.

---

##  Funcionalidades
- Cadastro de usuários (alunos, professores, cidadãos).
- Cadastro de cursos e campanhas educativas.
- Inscrição e participação em cursos.
- Emissão de certificados digitais.
- Relatórios automáticos de participação.
- Módulo de denúncias de cyberbullying e discurso de ódio.
- Notificações educativas para os usuários.
